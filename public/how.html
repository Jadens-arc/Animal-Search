How Does it work

Trie

A Trie Data Structure is a way of organizing words so they can easily be searched through letter-by-letter

In a Trie each letter word is placed in it's own Node, each Node is a basic object with only a few properties


{
"value": can be any of the letters in the alphabet,
"children": a list of what letters come after this one across all words,
"isWord": a boolean[True or False] which communicates whether this is the final letter in a word
}
